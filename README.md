# Опис проєкту
### Ця програма використовує базовий функціонал введення/виведення даних через консоль, що б створити процес нескінченної міні гри з лічильником досвіду. 

# Принцип роботи
Кожен повтор програма генерує випадкову число від 0 до 10 а в гравця є 4 спроби вгадати його. Кількість отриманого досвіду дорівнює кількості невикористаних спроб помноженого на 10, а у випадку програшу - віднімається 10.
